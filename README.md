# E-commerce Project

This is a sample e-commerce API project for testing AI code review workflows.

## Features
- User management
- Product catalog
- Order processing
- File operations

## Setup
```bash
npm install express mysql
node ecommerce-api.js
```

## API Endpoints
- GET /users/:id
- POST /products
- GET /orders/:userId

## Security Notes
⚠️ This code contains intentional security vulnerabilities for testing purposes.
Do not use in production!
